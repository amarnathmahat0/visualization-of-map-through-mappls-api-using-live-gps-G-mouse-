# Visualization of Map through Mappls API using Live GPS (G-Mouse)

This project visualizes real-time GPS data on a map using the **Mappls (MapmyIndia) API**. It captures live coordinates from a G-Mouse GPS receiver and displays them dynamically on a web-based interface.

## 🚀 Features
* **Real-time Tracking:** Integration with G-Mouse GPS for live coordinate updates.
* **Mappls API Integration:** High-quality map visualization with custom markers.
* **Dynamic Markers:** Automatically updates the marker position as the GPS coordinates change.
* **Node.js Backend:** Handles server-side logic and static file serving.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Backend:** Node.js, Express.js
* **Maps API:** Mappls API (MapmyIndia)
* **Hardware:** G-Mouse GPS Receiver

## 📂 Project Structure
* `server.js`: The main entry point for the Node.js server.
* `index.js`: Client-side logic for map initialization and marker updates.
* `navigation.html`: UI for the map navigation interface.
* `changing_marker_with_random_lat_and_long.html`: A testing module for simulating coordinate changes.

## ⚙️ Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/amarnathmahat0/visualization-of-map-through-mappls-api-using-live-gps-G-mouse-.git](https://github.com/amarnathmahat0/visualization-of-map-through-mappls-api-using-live-gps-G-mouse-.git)
   cd visualization-of-map-through-mappls-api-using-live-gps-G-mouse-
Install dependencies:

Bash
npm install
Mappls API Key:

Get your API keys from the Mappls Developer Admin Console.

Replace the placeholder keys in the HTML/JS files with your actual credentials.

Run the application:

Bash
node server.js
Open http://localhost:3000 (or your configured port) in your browser.
